lxc start (Nombre Contenedor)

lxc exec (Nombre Contenedor) bash

Creamos un directorio para pydio, nos posicionamos en la carpeta creada, descargamos pydio y lo extraemos

sudo mkdir pydio && cd pydio && apt install unzip && wget https://download.pydio.com/pub/cells/release/1.6.1/linux-amd64/pydio-cells-1.6.1-linux-amd64.zip && unzip pydio-cells-1.6.1-linux-amd64.zip && chmod u+x pydio-cells-1.6.1-linux-amd64.zip

Es importante destacar que es necesario instalar mysql, la más probable es que ya esté instalado, pero si no, usaremos:

sudo apt install mysql-server

Habilitamos el servicio e iniciamos mysql:

sudo systemctl enable mysql

sudo system start mysql

Entramos en mysql:

mysql -u root -p

Configuramos una base de datos con un usuario:

CREATE USER 'Usuario'@'localhost' IDENTIFIED BY 'Contraseña';

CREATE DATABASE cells;

GRANT ALL PRIVILEGES ON cells.* to 'Usuario'@'localhost';

FLUSH PRIVILEGES;

EXIT

Concedemos permisos de ejecución al binario:

chmod +x cells

chmod +x cells-ctl

Dependiendo del entorno que tengamos ejecutaremos cells o cells-ctl, en este caso usaremos cells-ctl, para ello:

./cells-ctl
